# customHtml5VideoWindowStyle
uniapp使用nvue原生子窗口自定义的一个Html5端的视频样式设置Demo(注意：只有在真机运行环境才能显示)

## 介绍
### vue2.0 + css + js基于uniapp进行开发的一个简单的html端的自定义视频播放控件DEMO,主要在subvue子窗口下对weex的渲染
